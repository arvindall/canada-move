# canada-move

Planning and evidence repository for the California → Canada move.

## Repository layout

- `canada-asset-agent/` — Phase 1 tooling for the Canada arrival-date brokerage holdings and FMV snapshot
- `relocation/` — move planning documents and checklists

## Where to start

- use `relocation/README.md` for the current move strategy, tax-sensitive timeline, houseplant import notes, and documentation priorities
- use `relocation/checklist.md` for the move task list
- use `canada-asset-agent/README.md` for the brokerage snapshot workflow

## Data and outputs

The brokerage subproject keeps raw inputs and generated outputs under `canada-asset-agent/data/`. Those directories remain ignored by Git, so keep the authoritative evidence pack locally and archive dated exports/snapshots outside version control when needed.
