# Burnaby Childcare Action Sheet
Use this sheet with `../checklist.md` when the childcare search becomes active.

## Search order
1. Start with licensed `30 months to school age`, `3–5`, and `Early Years 0–5` programs
2. Use Burnaby CCRR as the referral and filtering layer
3. Use the B.C. Child Care Map for licensed provider discovery
4. Apply directly to shortlisted operators
5. Use licensed `multi-age`, `in-home multi-age`, and `family child care` as bridge options if a centre spot is delayed

## Burnaby CCRR
YMCA BC Child Care Resource and Referral

- Web: https://www.ymcabc.ca/child-care-resource-referral
- Phone: 604-294-1109
- Email: ygv.ccrr@bc.ymca.ca
- Address: 4460 Beresford St, 3rd Floor, Burnaby, BC V5H 0B8

Ask for:

- full-day care
- June/July start
- Metrotown / Burnaby commute fit
- licensed family / multi-age bridge options

Important: CCRR provides referrals and search guidance, not guaranteed openings.

## Official map
- B.C. Child Care Map: https://maps.gov.bc.ca/ess/hm/ccf/

Use these filters first:

- City: Burnaby
- Program type: `30 months to school age`

Then separately check:

- `multi-age`
- `in-home multi-age`
- `family child care`

## What to ask every provider
- Which exact program would a child turning 3 in July join?
- What is the earliest realistic full-day start date?
- Is the waitlist open now?
- Are you in CCFRI or `$10 a Day`?
- What is the expected monthly fee after reductions?
- Can the child remain there through kindergarten, or is another transition likely?
- Are there extra conditions such as toilet training or age-by-December cutoffs?

## After the BC address is confirmed
- Re-submit or update CCRR with the actual building address
- Re-rank providers based on commute reality, not target neighborhood
- Search for 2–3 bridge options in licensed `multi-age`, `in-home multi-age`, or `family child care`

## StrongStart BC
StrongStart can be a useful parent-accompanied bridge while waitlists move.

- Bring: child's passport or birth certificate, plus lease agreement / BC address proof
- Result: your child gets a Personal Education Number (PEN)

## Employer check
Kids & Company has a corporate-partner workflow. Check the Infosys portal / benefits package before assuming any priority access.
